Welcome to Grey H@t Root the Box


Hello and welcome to Grey H@t's instance of the Root the Box project.

This project serves to help educate anyone and everyone who is interested in learning about the world of information security. From time to time, the application may have live CTF competitions going on (see Grey H@t's site for more information) but for all the rest of the time, this project will host educational exercises for learning the ins and outs of all topics related to the information security field.

So go ahead and hop in! Whether you're a total n00b or a skilled veteran, we hope that this project will contain sufficient material to increase your knowledge about the in-depth and ever-expansive world of information security.

To get started go ahead and REGISTER an account, or LOG IN if you already have an account!

For more information about our SPONSORS, the PROJECT, Root the Box, or how YOU can supply educational materials to the project, check out the 'About' section at the top menu bar!
